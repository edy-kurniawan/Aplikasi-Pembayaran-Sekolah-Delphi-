# Aplikasi-Pembayaran-Sekolah (Delphi)
Aplikasi Pembayaran Sekolah Sederhana menggunakan Delphi 7 dan database Mysql

# Fitur :
 1. Login
 2. Data Siswa
 3. Data Pembayaran
 4. Transaksi Pembayaran
 5. Data Tunggakan Pembayaran
 6. Laporan
